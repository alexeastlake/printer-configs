# 0.2mm Nozzle

Aftermarket nozzle used for miniatures and fine detail work.

## Specs

| | |
|---|---|
| **Diameter** | 0.2 mm |
| **Material** | Hardened steel |
| **Source** | Aftermarket |

## Use Cases

- Miniatures
- Fine detail

## Layer Heights

| Layer height | Use |
|---|---|
| 0.04 mm | Maximum detail, very slow |
| 0.08 mm | Standard miniature detail |
| 0.12-0.16 mm | Faster, still finer than 0.4mm allows |

## Calibration Status

Per-filament calibration values live in the filament files:

- [Creality Ender PLA+ Black](../filaments/pla-plus-creality-black.md) — calibrated
- [Creality CR-PETG Black](../filaments/petg-creality-black.md) — not yet calibrated

## Notes

- Prints take significantly longer than the 0.4mm - plan accordingly.
