# 0.4mm Nozzle

Default nozzle shipped with the Creality K1C. Used for general purpose tasks.

## Specs

| | |
|---|---|
| **Diameter** | 0.4 mm |
| **Material** | Stock K1C hardened (as shipped) |
| **Source** | Creality (OEM) |

## Use Cases

- Functional/mechanical parts
- Large decorative prints
- Anything where speed and layer adhesion matter more than fine detail
- Default

## Layer Heights

| Layer height | Use |
|---|---|
| 0.12 mm | Fine-ish detail, slower |
| 0.20 mm | Standard / default |
| 0.28 mm | Fast / draft |

## Calibration Status

Per-filament calibration values live in the filament files:

- [Creality Ender PLA+ Black](../filaments/pla-plus-creality-black.md) — PA and retraction not yet calibrated for this nozzle
- [Creality CR-PETG Black](../filaments/petg-creality-black.md) — not yet calibrated

## Notes

- Prints faster than the 0.2mm.
- Default Orca profiles for the K1C are tuned around this nozzle.
