# Creality K1C

## Specs

| | |
|---|---|
| **Model** | Creality K1C (2025) |
| **Kinematics** | CoreXY |
| **Enclosure** | Fully enclosed, removeable lid |
| **Hotend** | All-metal, direct drive extruder |
| **Bed** | Smooth PEI (flexible, magnetic) |
| **Build volume** | 220mm × 220mm × 250 mm |
| **Firmware** | Stock |

## Mods/Aftermarket

- **0.2mm hardened steel nozzle** - for miniatures and fine detail.

## Nozzles

| Nozzle | Use case | Typical layer heights |
|---|---|---|
| 0.4mm | General purpose, functional | 0.12-0.28 mm |
| 0.2mm | Miniatures, fine detail | 0.04-0.16 mm |

## Slicer

- **Primary:** OrcaSlicer
- **Secondary:** Creality Print - only used as a monitoring application

## Enclosure

Lid off for all current filaments. See individual [filament files](../filaments/) for material-specific notes.

## Maintenance Notes

- Clean PEI with IPA periodically; avoid touching the bed with bare fingers (skin oils kill adhesion).
- See [PETG filament notes](../filaments/petg-creality-black.md#notes) for bed adhesion warnings specific to that material.
