# Profiles

Exported OrcaSlicer profiles, versioned alongside the notes.

## Structure

Organise exports into subdirectories:

- `printer/` — machine config per nozzle (e.g. `creality-k1c-0.4mm.json`)
- `filament/` — per-filament settings (e.g. `pla-plus-creality-black.json`)
- `process/` — per-combo slice settings (e.g. `0.20mm-standard-0.4nozzle.json`)

## When to update

Re-export after any calibration run or settings tweak you want to keep. Keep values in the matching filament markdown file in sync.
